One-time setup

This site triggers a GitHub Actions workflow to fetch tender data server-side (the government API blocks direct browser requests). It needs a GitHub token with Actions: write permission — created once, saved in your browser only.

Create token: github.com/settings/tokens → New fine-grained token → select this repo → Repository permissions → Actions = Read & Write → Generate token.

Token is stored only in your browser (localStorage). Never sent anywhere except api.github.com.
